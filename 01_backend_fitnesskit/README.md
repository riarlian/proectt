## Описание

В данном репозитории содержится backend приложение для управления задачами (Todo API).

Приложение реализует REST API и позволяет:

- создавать задачи
- получать список задач
- обновлять задачи
- удалять задачи

## Используемые технологии

- Python
- Django
- Django REST Framework
